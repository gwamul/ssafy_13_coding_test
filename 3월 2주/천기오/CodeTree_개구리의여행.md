메모리 사용량 - 23MB
날짜 / 시간 - 2026-03-09 / 11:04

문제 유형: 다익스트라 + 분기 조건

문제 전략
1. 결론적으로는 시작점에서 목적지까지 최단거리를 찾는 문제. 그러나 4가지 이동경로 외에 점프력 올리기/내리기 라는 행동이 추가되었고, 올리기는 cost가 1이 아니므로 bfs는 사용 불가능.
   - 다익스트라 문제 
2. 문제를 잘 읽기만 했다면 정말 쉬운 문제. 이정도면 A형보다 좀 더 어려운 수준? 제대로 안 읽어서 오래 걸리고 한번 틀리기까지 했다.
3. 보드에 저장되는 값은 총 3가지다. 안전한 돌'.' 미끄러운 돌'S' 천적이 있는 돌'#'. 점프를 했을 때 안전한 돌로만 이동할 수 있고, 미끄러운 돌로 착지는 불가능하다. 점프 경로 사이에 천적이 있다면 이동 불가능.
4. 그렇다면 우리가 할 수 있는 행동은 총 6가지다. 점프력 올리기, 점프력 내리기, 상하좌우 현재 점프력으로 이동하기. 점프력 내리기는 1~k-1까지 전부 가능하므로 이 점 주의. 이것때문에 틀렸다.
5. 거리는 3차원으로 저장해야한다. dist[x][y][p] = (x,y)까지 p의 점프력으로 올 수 있는 최단시간. 검사하는 값이 이보다 크거나 같다면 가지치기가 가능하다.
6. 문제를 잘 읽고 요구사항대로 작성했다면 쉽게 맞을 수 있는 문제이다. 4방향 이동의 경우 점프 사이에 적이 있는 지 확인해야한다. 이 조건을 못읽고 지나쳐서 문제 푸는 데 좀 오래 걸렸다.
